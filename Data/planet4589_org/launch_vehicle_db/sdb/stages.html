1.5: The stages database

The Stages file gives basic information for all of the stages making up the launch vehicles in the LV file. The reader is again cautioned about the approximate nature of the numerical values in this file.

The file contains for each stage:

  • Stage name

  • Stage family

  • Manufacturer

  • Alternate name

  • Length, diameter (m)

  • Full mass (tonne)

  • Dry mass (kg)

  • Typical thrust (kN)

  • Typical burn duration (s)

  • Main engine name

  • Main engine multiplicity

  • Stage type (this field should be ignored in the current release)

The main engine name has been made up (e.g. RD R-29 (1) for the first stage of the R-29) for cases where no name for the engine is available. A separate file of engines is perennially in preparation.
